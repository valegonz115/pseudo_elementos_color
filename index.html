♥ Pseudo Elementos ♥

❝active❞

Este enlace cambiará a color amarillo mientras hace clic en él.

🎔 Representa un elemento (como un botón) que el usuario está activando.

❝any-link❞

Enlace externo

🎔 La pseudo-clase :any-link de CSS representa a un elemento que actúa como el ancla origen de un hipervínculo independientemente de si ha sido visitado.

❝focus❞



🎔 La pseudo-clase :focus CSS representa un elemento que ha recibido el foco. Generalmente se activa cuando el usuario hace clic, toca un elemento o lo selecciona con la tecla "Tab".

❝focus-visible❞


🎔 La pseudo-clase -focus-visible se aplica mientras un elemento coincide con la pseudo-clase -focus y el UA (Agente de Usuario) determina mediante heurística que el foco debe hacerse evidente en el elemento.

❝focus-within❞


🎔 La pseudo-clase representa un elemento que ha recibido el foco o que contiene un elemento que ha recibido el foco.

❝fullscreen❞

🎔 La pseudo-clase -fullscreen de CSS representa un elemento que se muestra cuando el navegador está en modo de pantalla completa.

❝hover❞

Intenta pasar el mouse sobre este enlace.

🎔 La pseudo-clase :hover de CSS coincide cuando el usuario interactúa con un elemento con un dispositivo señalador, pero no necesariamente lo activa.

❝indeterminate❞

🎔 Cuyo estado sea indeterminado.

❝in-range❞

    Los valores entre 1 y 10 son válidos.

🎔 La pseudo-clase -in-range de CSS representa un elemento cuyo valor actual se encuentra dentro de los límites de rango especificados por los atributos min y max.

❝invalid❞

🎔 La pseudo-clase :invalid de CSS representa cualquier elemento ... cuyos contenidos no se puedan validar

❝lang❞

🎔 La pseudo-clase es utilizada para modificar elementos en función del idioma en el que se determina que están.
